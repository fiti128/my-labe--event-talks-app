# My Awesome Event Talks App

This is a single-page application that displays the schedule for a one-day technical event.

## Features

- View the entire event schedule with timings.
- Search for talks by category.

## Tech Stack

- Node.js (for data generation)
- HTML, CSS, JavaScript

## How to Run

1.  Open the `index.html` file in your browser.
2.  Alternatively, you can run a simple web server to serve the file. For example, using Python:
    ```bash
    python3 -m http.server 8000
    ```
    Then open `http://localhost:8000` in your browser.
